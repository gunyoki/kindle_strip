# KindleStrip

Strip SRCS records from .mobi

## Installation

Add this line to your application's Gemfile:

    gem 'kindle_strip'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kindle_strip

## Usage

    $ kindle_strip input.mobi output.mobi
